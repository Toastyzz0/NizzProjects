# Nizz - Manage your Projects
A app that reminds you about your projects and allows you to save them for later. Based on [Milanote](https://milanote.com).

> [!WARNING]
> This is my first project made in React Native so, my code might be bad.

## About:
Nizz allows you to add projects that you're working on and manage things to give you reminders or more. There's also webhooks that allows you to connect to Github through the app to see any commits for the Repository.

> [!CAUTION]
> Anything could or will be changed. Since this is just a test project to test my skills, Most things like Webhooks and looking at commits might not happen.


## How to Download:
Nizz is only available on Android and Desktop (Depending). Download them from the [releases](https://github.com/Toastyzz0/NizzProjects/releases) page. Note that if there's nothing there, its not done yet so be patient until it's done. >:(

## How to Clone:
Go to your command prompt and type:

```
mkdir nizzprojects && cd nizzprojects
```
Then, type:

```
git clone https://github.com/Toastyzz0/NizzProjects.git
```

Then that's all I know. Bye!
